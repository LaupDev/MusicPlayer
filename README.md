# MusicPlayer

This project was created based on [Philipp Lackner's YouTube video series](https://youtube.com/playlist?list=PLQkwcJG4YTCT-lTlkOmE-PpRkuyNXq6sW)

### What was used:
* MVVM
* Kotlin
* Kotlin Coroutines
* ExoPlayer
* Hilt
* Firebase Cloud Firestore
* Firebase Storage
* Navigation component
* View Binding
* Glide
* Material Design
* Timber
